# 🐦 Flappy Bird

A browser-based revival of the iconic *Flappy Bird* arcade game — tap, dodge, and survive for as long as you can. Built with pure *HTML, **CSS, and **JavaScript*, all packed into a single file with zero dependencies.

---

## ✨ Features

- ⚡ Fluid, lag-free controls that feel great to play
- 🎲 Procedurally generated pipes — no two runs are the same
- 📈 Live score counter to track your personal best
- 🎯 Accurate collision detection for fair, precise gameplay
- 🔁 One-click restart to get back in the action immediately

## ⚙️ Setup & Installation

*Step 1 — Clone the repo:*

bash
git clone https://github.com/UsmanDanial04/Flappy-Bird-Game.git
cd Flappy-Bird-Game


*Step 2 — Play:*

Just open index.html in any modern browser. That's it — no build tools, no installs, no fuss.

---

## 🕹️ Controls

| Action | Input |
|---|---|
| Flap wings | Spacebar or screen tap |
| Restart after game over | Click the restart button |

Navigate the bird through pipe gaps, avoid hitting anything, and push your score as high as possible!

## 🔧 Want to Tinker?

The game is easy to customize — everything lives inside index.html:

- *Swap the bird* → Replace assets/bird.png with any image you like
- *Speed things up* → Find the speed variable and crank it higher
- *Adjust difficulty* → Play with pipe gap sizes or tweak the gravity value
